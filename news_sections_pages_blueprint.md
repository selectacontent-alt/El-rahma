# مخطط تقسيم صفحات أقسام موقع الأخبار

> **المطلوب:** تحويل عناصر المنيو الحالية إلى أقسام أعمق مثل: التقنية، الذكاء الاصطناعي، الاقتصاد وسوق الأسهم، السوشيال ميديا… مع جعل كل صفحة قسم لها شكل بصري يعبر عن عنوانها، مع الحفاظ على هوية الموقع.  
> **النظام البصري الأساسي:** Light Mode  
> **ألوان الهوية:**  
> - Primary: `#9d027c`  
> - Secondary: `#ffbc01`  
> - Background: `#ffffff`  
> - Soft Background: `#f7f7f9`  
> - Text: `#161616`  
> - Muted Text: `#6b6b6b`

---

## 1. شكل المنيو الجديد المقترح

بدل المنيو الحالي:

```txt
الرئيسية | أخبار الشركة | أخبار السوق | مقالات | دراسات حالة | فيديوهات
```

يكون المنيو الجديد أكثر وضوحًا للـ SEO والزائر:

```txt
الرئيسية
أخبار الشركة
تقنية
ذكاء اصطناعي
الاقتصاد وسوق الأسهم
سوشيال ميديا
تسويق رقمي
دراسات حالة
مقالات
فيديوهات
```

### نسخة مختصرة لو المنيو طويل

```txt
الرئيسية
أخبار الشركة
تقنية
ذكاء اصطناعي
اقتصاد وأسهم
سوشيال ميديا
مقالات
فيديوهات
المزيد
```

وتحت "المزيد":

```txt
دراسات حالة
تسويق رقمي
ريادة أعمال
أمن سيبراني
تقارير
```

## 3. القاعدة العامة لكل صفحة قسم

كل صفحة قسم لازم يكون لها نفس الهيكل الأساسي، لكن بتصميم مختلف يعبر عن القسم.

```txt
Header
Navigation
Category Hero
Featured Section
Latest Articles
Special Block حسب نوع القسم
Sidebar
Pagination
Footer
```

### الهيكل العام

```txt
┌──────────────────────────────────────────────┐
│ Header + Navigation                          │
├──────────────────────────────────────────────┤
│ Breadcrumb                                   │
├──────────────────────────────────────────────┤
│ Category Hero                                │
│ عنوان القسم + وصف قصير + صورة/أيقونة رمزية │
├──────────────────────────────────────────────┤
│ Featured News / أهم الموضوعات                │
├──────────────────────────────────────────────┤
│ Main Content Grid + Sidebar                  │
├──────────────────────────────────────────────┤

├──────────────────────────────────────────────┤
│ المزيد من المقالات + Pagination              │
├──────────────────────────────────────────────┤
│ Footer                                       │
└──────────────────────────────────────────────┘
```

## 4. صفحة قسم التقنية

### رابط الصفحة

```txt
/technology
```

أو:

```txt
/تقنية
```

### فكرة التصميم

صفحة التقنية تكون شكلها سريع وحديث، فيها إحساس بالتطور والديجيتال.

### الهوية البصرية للقسم

- خلفية بيضاء مع أشكال خفيفة جدًا مثل خطوط Circuit أو مربعات Pixel.
- استخدام البنفسجي `#9d027c` في العناوين.
- استخدام الأصفر `#ffbc01` كعلامة Highlight على الأخبار المهمة.
- يمكن إضافة لون مساعد خفيف جدًا مثل الأزرق السماوي فقط في الرسومات الصغيرة وليس كهوية أساسية.

### Hero Section

```txt
┌────────────────────────────────────────────┐
│ تقنية                                      │
│ آخر أخبار التكنولوجيا، التطبيقات، الأجهزة، │
│ والشركات التقنية في مكان واحد.             │
│                                            │
│ [آخر أخبار التقنية] [مراجعات] [أمن سيبراني]│
└────────────────────────────────────────────┘
```

### شكل المحتوى

```txt
┌───────────────────────────┬────────────────┐
│ خبر رئيسي كبير             │ أخبار قصيرة     │
│ صورة تقنية كبيرة           │ 4 كروت صغيرة    │
└───────────────────────────┴────────────────┘
```

### بلوك خاص بالقسم

```txt
أحدث المراجعات التقنية
[موبايل] [تطبيق] [منصة] [أداة]
```

### عناصر تعبر عن التقنية

- أيقونات Microchip.
- كروت بأطراف دائرية.
- خطوط Grid خفيفة.
- صور أجهزة، شاشات، تطبيقات، Software Dashboards.
- Tags مثل: تطبيقات، أجهزة، شركات تقنية، أمن.

نستخدم Light Mode مع تأثيرات زجاجية بسيطة.

```txt
│ أخبار وأدوات وشروحات AI وتأثيرها على       │
│ الأعمال والتسويق وصناعة المحتوى.           │
│                                            │
│ [أدوات AI] [شروحات] [تطبيقات في الأعمال]  │
└────────────────────────────────────────────┘
```

### شكل المحتوى

```txt
┌──────────────────────────────┐
│ خبر رئيسي: أداة AI جديدة      │
├──────────────┬───────────────┤
│ شروحات AI    │ أخبار AI       │
├──────────────┴───────────────┤
│ أدوات مختارة هذا الأسبوع      │
└──────────────────────────────┘
```

### بلوك خاص بالقسم

```txt
أدوات AI ننصح بها
┌────────────┬────────────┬────────────┐
│ اسم الأداة │ الاستخدام  │ رابط الشرح │
└────────────┴────────────┴────────────┘
```

### عناصر تعبر عن الذكاء الاصطناعي

- أشكال Neural Network خفيفة جدًا في الخلفية.
- Icons: Brain، Bot، Spark، Prompt.
- كروت اسمها: "أداة اليوم"، "شرح سريع"، "تطبيق عملي".
- قسم صغير: "Prompt of the Week" لو الموقع يقدم محتوى تعليمي.

### محتوى مناسب للقسم

- أخبار أدوات الذكاء الاصطناعي.
- شروحات استخدام AI في الشركات.
- تأثير AI على التسويق.
- تطبيقات AI في خدمة العملاء.
- مقارنات بين أدوات AI.


### شكل المحتوى

```txt
┌────────────────────────────────────────────┐

│ EGX30 +0.4% | Gold -0.2% | USD ثابت        │
```

### بلوك خاص بالقسم

```txt
تحليل السوق اليوم

- توقعات عامة
```

### عناصر تعبر عن الاقتصاد والأسهم

- Mini Charts داخل الكروت.
- أرقام ونسب مئوية.
- سهم صاعد وسهم هابط.
- ألوان مساعدة محدودة:
  - أخضر للأخبار الإيجابية.
  - أحمر للأخبار السلبية.
  - لكن بدون الخروج عن هوية البنفسجي والأصفر.

### كروت الأخبار

```txt

عنوان الخبر
ملخص من سطرين
نسبة / رقم مهم
```

## 20. ألوان مساعدة لكل قسم مع الحفاظ على الهوية

> الألوان المساعدة تستخدم في الأيقونات والـ Background Shapes فقط، وليس بدل الهوية الأساسية.

```txt
تقنية: أزرق فاتح مساعد
ذكاء اصطناعي: بنفسجي فاتح / أزرق ناعم
اقتصاد وأسهم: أخضر للأرقام الإيجابية وأحمر للسلبية
سوشيال ميديا: Gradient خفيف من البنفسجي للأصفر
تسويق رقمي: أصفر واضح مع البنفسجي
دراسات حالة: بنفسجي رسمي مع أرقام كبيرة
فيديوهات: أصفر للـ Play Button
أمن سيبراني: أزرق داكن مساعد مع Shield Icons
ريادة أعمال: أصفر دافئ مع صور فرق عمل
```
